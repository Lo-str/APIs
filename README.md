Lets finish our final project!

In this assignment you are going to create a registration and login page for your minigame.

Create a registration page

Tasks:

Create a separate page for your registration

It should have username and password as inputs

Add the credentials in the localStorage, one for username and one for password

Feel free to add some requirements as well, for example (must contain a symbol, must contain at least 8 characters and so on)

Create a login page

Tasks:

Create a login page where the user can enter in their login credentials

Make sure that the login information is correct when the user clicks on login

If all the information is correct, the user should be directed to the game page

Feel free to add any amount of CSS
